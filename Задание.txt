Задания со звездочкой повышенной сложности. Выполняются по желанию.
Для решения заданий Jquery не используем. Никаких дополнительных библиотек скачивать не надо. Все уже подключено.
В адаптивной верстке сайта используются единицы измерения vw. Менять их на другие единицы измерения нельзя.

Представьте, что вам пришел заказ доработать сайт. Вам скинули архив с сайтом и список изменений.


///////// Что нужно сделать на сайте: 

1. Фиксированная шапка сайта, которая остается при пролистывании сайта

2. Логотип должен быть ссылкой на главную страницу

3. По задумке дизайнера изображение на главной странице должно быть квадратное, 
   но загружено было не квадратное. Из-за этого произошло искажение изображения на сайте.
   Нужно исправить это

4. После заголовка "Создание сайтов под ключ" надо добавить кнопку "Заказать сайт"
   Кнопка должна открывать модальное окно с формой заявки. JS функция для открытия модального окна уже сделана в файле script.js.

5. В блоке "Наши услуги"  кнопки "Заказать" так же должны открывать модальное окно с формой заявки

6. У сайта нет Фавиконки

7. Форма в модальном окне не оформлена стилями. Нужно оформить ее на свой вкус

8*. При клике по кнопке "Заказать" в блоке "Наши услуги" надо с помощью js получить название услуги и 
    добавить его в скрытое поле в форме

9*. Сделать отправку формы без перезагрузки страницы. Выводить сообщение об успешной отправке формы. 
    Создать php скрипт, который будет принимать данные из формы и возвращать положительный ответ

10*. Предложить структуру базы данных, в которой будут следующие данные:  Услуги, Заказы, Клиенты


/////////// Задания для компьютерной версии сайта:

1. Плавное подчеркивание пунктов меню при наведении на них

2. Номер телефона должен быть выделен рамкой и при наведении менять фон

3. Карточки услуг позиционировать по центру. При добавлении новой услуги, ее карточка должна автоматически переноситься
   на новую строку, а не выходить за область экрана.


////////// Задания для мобильной и планшетной версии сайта:

1. Сделать бургер меню для сайта. Верстка кнопки уже сделана. Задача сверстать меню и открывать его по клику на кнопку.
   Кнопку бургер-меню надо поменять местами с номером телефона при этом не меняя html-разметки

2. На мобильной версии - главная картинка должна быть первой, а заголовок вторым. Это надо сделать так же, не меняя html-разметку
